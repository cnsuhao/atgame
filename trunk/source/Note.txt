///////////////////////////////////////////////////////////////
// 代码习惯
///////////////////////////////////////////////////////////////
+ 成员变量以 "_" 开头 如 _parent;
+ 提倡重载类的虚函数时要使用atgBase.h里面的 OVERRIDE 宏后缀. 



///////////////////////////////////////////////////////////////
// 系统内容
///////////////////////////////////////////////////////////////
已实现:
+ game main loop 与系统无关, 目前支持(windows , android)
+ input system 捕捉了鼠标,手指事件.提供了简单接口.
+ timer 实现了window, linux无关的时间系统.
+ logger 很简陋的日志输出功能.
+ android 自动NDK编译. 自动ANT安装到手机运行. 
+ function profile. 分析功能内来自bullte的profile.


///////////////////////////////////////////////////////////////
// 图形内容
///////////////////////////////////////////////////////////////
已实现:
+ opengl, opengl es, directx9 封装统一接口.
+ mesh import. (fbx, mdx) 模型导入可以很快绘制出制作好的模型
+ skin animation. 基于CPU端的动画,骨骼分离的动画系统.
+ material 非常简陋的材质支持. 不过底层基本上完成,上层只需要完备而已.
+ camera 功能齐全的支持欧拉角,四元数,矩阵之间任意切换的的相机.  
+ shader library factory. 通过统一接口创建shader.  
+ el UI system. 移植了一个简单改造的CEGUI系统.



未实现:

+ 添加ffp batch(批量)绘制器.
+ 视椎体裁剪
